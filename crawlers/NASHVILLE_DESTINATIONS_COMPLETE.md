## Usage Examples

### Query All Nashville Honky-Tonks
```python
import db
client = db.get_client()

honky_tonks = client.table('venues').select('*') \
    .eq('city', 'Nashville') \
    .contains('vibes', ['honky-tonk']) \
    .execute()

# Returns: Tootsie's, Robert's, Layla's, Acme, Legends, Sutler
```

### Query All Coffee Shops
```python
coffee_shops = client.table('venues').select('*') \
    .eq('city', 'Nashville') \
    .eq('venue_type', 'coffee_shop') \
    .execute()

# Returns: 15 coffee shops including Barista Parlor, Frothy Monkey, Crema, etc.
```

### Query Music Venues by Neighborhood
```python
east_nash_music = client.table('venues').select('*') \
    .eq('city', 'Nashville') \
    .eq('neighborhood', 'East Nashville') \
    .eq('venue_type', 'music_venue') \
    .execute()

# Returns: The Basement East, The 5 Spot, Drkmttr, Five Points Pizza
```

### Query LGBTQ+ Venues
