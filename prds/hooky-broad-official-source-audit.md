# Hooky Broad Official-Source Audit

**Portal:** `hooky`
**Surface:** `consumer`
**Status:** First broad pass
**Last Updated:** 2026-03-10

---

## Executive Summary

This pass broadens the Hooky program map beyond the first premium-provider audit.

The result is clear:

1. Hooky already has meaningful strength in premium institution-led family programming
2. The biggest depth gaps versus the market are now confirmed as:
   - school-hosted camps
   - long-tail STEM and specialty enrichment
   - sports / swim / movement providers
   - local arts/music/dance operators
3. Many of these gaps do have usable official-source surfaces
4. That means Hooky can continue research-first and still build toward competitive depth without copying competitor directories

This pass is still discovery and source validation. It is not a recommendation to build all of these crawlers immediately.

---

## Current Strategic Position

Hooky’s strongest existing or already-proven source classes remain:

- museums and cultural institutions
- arts camps and theater programming
- coding and maker-style enrichment
- a few strong swim/program platforms

That gives Hooky a premium core.

What competitors add on top is not better structure. It is a wider provider universe that includes:

- independent school summer programs
- local STEM operators
- local sports and movement providers
- neighborhood arts/music schools
- municipal and civic family-program layers

To feel complete, Hooky needs to map those layers even if it only builds a subset first.

## Market Implications

### 1. Hooky can now justify a broader official-source roadmap

The market no longer looks like:

- a handful of premium institutions

It looks like:

- premium institutions
- school-camp networks
- franchise enrichment
- long-tail local specialists
- municipal and civic programs

That is a very different sourcing challenge.

### 2. School camps should be treated as a full category, not one-off sources

This category is too large to treat opportunistically. It needs a category-level plan.

### 3. Swim is the cleanest first move inside sports/movement

Swim has more visible recurring-program structure than most youth sports categories.

### 4. STEM and chess are stronger than they first looked

Club SciKidz, Camp Invention, Kid Chess, and theCoderSchool together suggest a much more robust STEM/specialty layer than Hooky currently surfaces.

### 5. Arts can become a genuine signature category

Hooky is already good here. The right move is to keep widening the category with high-quality local operators.
