# Yonder Artifact Candidate Sheet

**Parent docs:** `prds/034s-yonder-artifact-landmark-quest-research.md`, `prds/034t-yonder-artifact-source-and-linking-strategy.md`, `prds/034-yonder-adventure-portal.md`  
**Status:** Draft  
**As of:** 2026-03-12  
**Purpose:** Establish the first real Yonder artifact candidate slate, grouped by type and parent destination, so artifact/quest work can move from abstract concept to actual content planning.

---

## 1. Research Read

This first pass does not try to finish the entire `100+` artifact universe.

It establishes a **strong Phase 1 slate** large enough to:

- prove that the concept has range
- show which source families are producing the best candidates
- draft a real quest slate
- reveal where the remaining density work should go next

Current expanded slate size:

- `144` artifact candidates

Source mix:

- Atlas Obscura for urban oddities, landmarks, outsider art, and unusual history
- Atlanta Trails for waterfalls, overlooks, fire towers, and hike-payoff artifacts
- Explore Georgia for natural wonders, scenic landmarks, and statewide "hidden gem" framing
- existing Yonder destination graph for parent anchors and confidence around geography

This is enough to start real launch curation and quest packaging.

---

## 2. Candidate Fields

Each artifact candidate is tagged here with:

- `artifact_type`
- `region`
- `likely_parent`
- `source_family`
- `quest_fit`

`quest_fit` is a quick editorial signal:

- `high` = clearly quest-worthy
- `medium` = useful support candidate
- `low` = likely better as a standalone oddity/landmark than a quest anchor

## 4. Parent-Destination Read

The current slate supports the expected parent-child model:

- waterfalls and overlooks mostly attach to existing Yonder destinations
- river-access artifacts often attach to support nodes or access nodes
- urban oddities are more likely to behave like semi-standalone artifacts inside an Atlanta cluster
- a minority of statewide outsider-art and oddity landmarks may warrant standalone destination treatment later

That is a good sign.
It means Yonder can likely seed artifacts without exploding the destination model.

---

## 5. Strongest Quest Material In This Slate

The strongest quest material already visible in the expanded sheet is:

- North Georgia waterfalls
- fire towers and summit lookouts
- Chattahoochee river access points
- Atlanta urban oddities / outsider art
- Georgia stone and geology landmarks
- coastal wild-edge landmarks
- beginner-friendly big-payoff artifacts

Those are the quest families most worth drafting first.

---

## 6. What This Sheet Still Needs

This sheet is strong enough to start quest design, but it is not yet the full launch universe.

The next expansion pass should add:

- more dog-friendly artifacts and easy overlooks
- more central Georgia river landmarks
- more cabin-adjacent or campground-adjacent artifact clusters
- more direct parent-child mapping into the seeded Yonder weekend set

That is how the slate gets from `144` to a true launch backlog with clear priority tiers.
