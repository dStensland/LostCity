# Yonder Launch Artifact Slate

**Parent docs:** `prds/034u-yonder-artifact-candidate-sheet.md`, `prds/034v-yonder-quest-slate.md`, `prds/034w-yonder-artifact-reference-contract.md`  
**Status:** Draft  
**As of:** 2026-03-12  
**Purpose:** Narrow the expanded artifact universe into a launch-ready shortlist that can support the first five Yonder quests without waiting for a full artifact platform.

---

## 1. Strategic Read

Yonder does not need to launch with the full `144`-item artifact slate behaving like first-class product objects.

It needs a smaller set that is:

- clearly worth going to
- easy to explain
- visually compelling
- geographically balanced
- aligned with the strongest launch quests

The right launch posture is:

- `20-30` launch-tier artifacts
- a wider support slate behind them
- external guide references for context and depth

---

## 2. Launch-Tier Selection Criteria

An artifact should make the launch tier when it has most of these qualities:

- strong payoff
- strong photo/social legibility
- clean parent-destination fit
- good source confidence
- low explanation burden
- useful quest role

Artifacts can still stay in the broader backlog if they are:

- more obscure
- weaker on access confidence
- too niche for first-time users
- better as support nodes than headline stops

## 5. Source Pattern For Launch-Tier Artifacts

Each launch-tier artifact should ideally have:

- `1` official logistics source
- `1` Yonder-authored blurb
- up to `1` guide-depth source
- up to `1` story source

Typical combinations:

- waterfall artifact
  - official park page
  - Atlanta Trails
- urban oddity
  - official or parent destination page if one exists
  - Atlas Obscura
- statewide scenic landmark
  - official destination page
  - Explore Georgia

This keeps the experience helpful without turning pages into source dumps.
