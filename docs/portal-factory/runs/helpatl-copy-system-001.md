# HelpATL Copy System 001

- Date: 2026-03-07
- Scope: consumer-facing copy for `helpatl`
- Goal: sound like a civic action guide, not a generic events app

## Voice Pillars

1. Actionable: every block should imply next step.
2. Clear: plain-language civic terms over jargon.
3. Trustworthy: concise source/freshness context at decision points.
4. Time-aware: emphasize windows and deadlines.

## Do / Don't

Do:
- "Join", "Track", "Attend", "Sign up", "Before"
- "City Council", "County Commission", "Board of Education"
- "Next 7 days", "This week", "Upcoming"

Don't:
- "Vibes", "Scene", "Trending now" as primary framing
- abstract engagement words without action verbs
- overly long explanatory paragraphs in cards

## Portal-Level Language Contract

Primary brand name:
- `HelpATL`

Core promise line:
- "Follow civic groups and volunteer opportunities across Atlanta."

Top nav labels (recommended):
1. `Act`
2. `Calendar`
3. `Groups`

## Key Surface Copy

### Feed Hero

Headline options:
1. "Your Civic Week in Atlanta"
2. "What Needs You This Week"
3. "City, County, School Board, Service"

Subheadline:
- "Join groups to personalize meetings, agendas, and volunteer opportunities."

Primary actions:
1. `Join Groups`
2. `See This Week`
3. `Volunteer Now`

### Interest Channels Section

Section title:
- `Join Groups`

Section subtitle:
- Default: "Follow city, county, school board, and civic groups."
- If joined: "{n} groups joined"

Button labels:
- `Join`
- `Joined`
- `Saving...`

### Groups Page

Page title:
- `Groups`

Page description:
- "Follow city, county, school board, and topic groups to keep your civic feed relevant."

Preset filter labels:
1. `All`
2. `City + County`
3. `School Board`
4. `Volunteer`
5. `Community`

Search placeholder:
- "Search groups (city, county, school board, volunteer...)"

### Event/Card Reason Labels

Prefix:
- `Matched`

Examples:
1. `Matched: City of Atlanta Government`
2. `Matched: Volunteer Opportunities`
3. `Matched: School Board Watch`

Source trust line:
- `Source: {source_name}`

Freshness line:
- `Updated {relative_time}`

## Microcopy For Empty/Edge States

No groups found:
- "No groups match your filters."

No matches yet:
- "No matched civic items yet. Try joining more groups."

Disabled/error state:
- "Groups are temporarily unavailable. Please try again."

## Admin UI Copy (HelpATL context)

Channels panel lead:
- "Define civic follow groups and matching rules."

Quality warning (no matches):
- "{channel_name} has subscribers but no matches. Check source coverage and rules."

Schedule helper:
- "Daily refresh keeps group matches current for civic timelines."

## Copy QA Checklist

1. Action verb present in all primary CTAs.
2. Civic nouns are concrete (city/county/board/volunteer).
3. No discovery-first slang in hero/top modules.
4. Reason/source context appears where users decide to click.
5. Mobile strings fit without truncating critical meaning.
