Design System Summary

  • Typography: 1.25 ratio scale with optimized line-heights for each use case
  • Spacing: Letter-spacing ranges from -0.02em (headings) to 0.05em (badges)
  • Colors: Cream (#F5F5F3) dimmed from pure white for reduced eye strain
  • Shadows: Multi-layer shadows with inner rim-light for premium depth
  • Borders: Subtle rgba(255,255,255,0.06) instead of harsh solid colors
  • Cards: Hover lift effect with smooth transitions
