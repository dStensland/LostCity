# Emory Consumer Portal Comps

Static visual comps for the consumer portal direction:

1. `golden-flow-hospital-now.html` - Single flagship hospital-now decision flow (elite demo target).
2. `hospital-hub.html` - Simple care front door + local activity feed.
3. `concierge.html` - Stay, food, and support planning around care.
4. `community-hub.html` - "Keeping Atlanta Healthy" community experience.

Open `index.html` in this folder for quick navigation.

These are pre-implementation direction comps meant to align on strategy, UX tone, and information architecture.
