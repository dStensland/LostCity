Why This Works

Breaking down the design decisions that make this mark effective.

LOST CITY

Design Strengths

  • Solid vs. Outline contrast — Creates visual tension and memorability. The filled "LOST" grounds the mark while the outlined "CITY" feels open, explorable.
  • Stacked layout — Compact, works as a square-ish icon. The vertical read is deliberate: you're lost first, then you find the city.
  • Coral pink (#FF6B7A) — Warmer and more alive than amber. Stands out without being aggressive. Works on dark and light.
  • Bold condensed type — Confident, urban, slightly editorial. Reads well at small sizes despite the tight tracking.
  • White outline — Clean, modern. The stroke weight matches the visual weight of the fill, creating balance.

Recommendations

Next steps for implementing this identity.

1. Lock in the Primary Mark

Use the coral (#FF6B7A) + white outline version as the default everywhere. Create SVG versions at multiple sizes (16px, 32px, 64px, 128px, 512px).

2. Update the Landing Page

Implement the "Coral Glow" concept — it's minimal but alive. Update the tagline to "Find what's happening tonight" for more urgency.

3. Adopt Bebas Neue + DM Sans

Bebas for the logo and section headers, DM Sans for everything else. Keep the current monospace for times/dates — it works well.

4. Replace Amber with Coral

Swap #F5A623 → #FF6B7A across the site. The coral is warmer, more distinctive, and aligns with the new mark.

5. Create Dark/Light Icon Variants

Ship the dark background icon as primary. Have the coral background version ready for light mode or contexts where dark doesn't work.

6. Add Subtle Glow Effects

Use the coral glow sparingly — on the landing hero, on featured events, on hover states. Don't overdo it, but let it add warmth.
