Animation System

Every interaction gets its moment. CSS-only where possible (keyframes, transitions), JS for the glitch effects and counter rolls. Performance-first — no janky repaints.

Key Design Elements

Corner Brackets

HUD targeting reticle on top-ranked images. Scales with rank importance.

Classification Labels

"CLASSIFIED // STUNT" headers. Typewriter font. Spy dossier energy.

Grid Background

Subtle tactical grid overlay on the page. Moves slowly. Surveillance camera feel.

Glitch Effects

Brief chromatic aberration + horizontal tear on transitions. Tasteful, not obnoxious.
