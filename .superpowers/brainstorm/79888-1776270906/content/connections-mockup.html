Design Principles
1. Every connection row is tappable — links to the connected entity's detail page
2. Counts are live — "14 upcoming events", "3 Atlanta dates" come from real data, not static text
3. Festival connections get accent treatment (gold border/bg) — they're time-sensitive and promotional
4. Social connections get coral accent — friends are the highest-value signal
5. Each entity type surfaces different connection types based on what makes sense — not a universal list
