Process

Phase 1: Design
1. Create design system in Pencil (tokens, spacing, section wrapper)
2. Design core sections as reusable components
3. Compose into full-page comps per entity type
4. Review web + mobile for each
5. Iterate until approved
Phase 2: Build
1. Core infrastructure (DetailLayout, types, traits, registry)
2. Section modules (one at a time, matching comps)
3. Identity zones (per entity type)
4. Orchestrators (wire it all together)
5. Delete old monoliths + dead code
